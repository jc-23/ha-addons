# JC-23 Home Assistant Add-ons

A small Home Assistant add-on repository.

## Add-ons

- **MAX! CUL Bridge** (`max2mqtt`) — bridges ELV MAX! thermostats to MQTT via a
  CUL/CUN stick. See [`max2mqtt/README.md`](max2mqtt/README.md) for details.

## Installing this repository

1. In Home Assistant, open **Settings → Add-ons → Add-on Store**.
2. Open the **⋮** menu (top right) and choose **Repositories**.
3. Add the URL:

   ```
   https://github.com/jc-23/ha-addons
   ```

4. The add-ons listed above now appear in the store and can be installed.
   They are built locally on your device (no prebuilt image is published).
